# 📊 DataWhisperer AI

An AI-powered CSV analysis application that allows users to upload datasets and ask questions in natural language. The application automatically generates and safely executes Pandas code using LangChain and Gemini AI, returning accurate answers, tables, and visualizations through a modern Streamlit interface.

---

## 🚀 Live Demo

🔗 https://datawhisperer-ai-yuhvp5pktqmantq4yccxdk.streamlit.app/

---

## ✨ Features

- 📂 Upload any CSV file
- 💬 Ask questions in plain English
- 🤖 AI generates Pandas code automatically
- 📊 Automatic tables and charts
- 📈 Dataset preview & statistics
- 📋 Dataset information panel
- 💡 Suggested analytical questions
- 🧠 Code explanation
- 🔒 Safe Python code execution
- 📝 Chat history
- 🎨 Modern Streamlit UI
